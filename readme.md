### Note:

You can run following command to rename plugin slug and namespace.

```bash
npm run rename -- --slug=<plugin-slug> --namespace=<plugin-namespace>

# For example:
npm run rename -- --slug=wp-temp-plugin --namespace=WP_TEMP_Plugin
```
